H5新增标签

1、section和article标签

section：把相关内容聚合到一起的div标签
article：作用与多个相关内容的聚合
aside：侧边栏，独立于主题内容部分，一般作用于广告位


页面布局的格式：
< header >< /header >
< nav >< /nav>
< main >
    < article >
    < section >< /section >
    < /article >
< /main >
< aside >< /aside >
< footer >< /footer >

2、form中新增标签


< input type=”tel” >文本框
< input type=”search” >搜索框，圆角格式
< input type=”url” >输入单个网址
< input type=”email” >输入电子邮件
< input type=”datetime” >定义日期，带有calendar和time控件
< input type=”date” >定义日期，带有calendar和time控件
< input type=”month” >定义月份，带有calendar控件
< input type=”week” >定义年份和周数，带有calendar控件
< input type=”time” >定义时分秒，带有time控件
< input type=”datetime-local” >定义日期有calendar和time控件
< input type=”number” >只能输入数字，移动端跳出数字输入框
< input type=”range” >一个可以拖动的进度条
< input type=”color” >定义颜色

input标签新增的属性

autofocus：页面加载时，光标自动对准
placeholder:：描述输入字段的提示信息
required：提供基本验证，比如验证是否输入为空
pattern：用于验证输入字段模式（正则表达式）
autocomplete：记录搜索历史，当前面输入相同时，自动提示历史


< input type=”text” value=”” required placeholder=”请输入密码” autofocus >

3、audio标签及属性

audio：定义声音、音乐、音频流
controls：显示控件，比如播放按钮，进度条，声音
autoplay：自动播放
loop：循环播放
preload：提前加载
src：引入地址
source：用于匹配浏览器所能播放的音频格式 


< audio controls autoplay loop >
< source src=”” title=”” >
< source src=”” title=”” >
< source src=”” title=”” >
< source src=”” title=”” >
< source src=”” title=”” >
< /audio >

4、video标签及属性

video：定义视频，比如电影片段及其他视频流
autoplay：自动播放
controls：显示控件，如播放按钮
height：定义播放器的高度
width：定义播放器的宽度
loop：循环播放
preload：提前加载
src：引入地址 


< video src=”” controls autoplay loop preload height=”200” >< /video >

5、hgroup、details、summary、mark、meter、progress标签

hgroup: 必须包括一个以上的h标签。
details：用于描述文档和文档的某个部分的细节，open属性定义details里面内容是否可见
summary：与details标签一起使用，作为details的标题，点击标题显示更详细的内容
mark：高亮文本，常用语搜索时候标记关键字
meter：定义范围的标量测量
progress：定义运行进度，进度条式样 


< hgroup >
    < h1 >< /h1 >
    < h2 >< /h2 >
< /group >
< details >
    < summary >目录1< /summary >
    < details >
        < summary >目录2< /summary >
        < details >
            < summary >目录3< /summary >
        < /details >
    < /details >
< /details >
< meter value=”5” min=”0”max=”10” >< /meter >
< progress value=”20” max=”100” >< /progress >

6、其他标签

1、abbr：表现一个缩写形式 


< abbr title=”People’s Republic of China” >PRC< /abbr >

2、address：一般用于footer,定义文章信息
3、bdi：设置一段文本,使其脱离父元素文本是指的方向
4、blockquote：一般引入长的语句,有固定的margin
5、code：作用于单行代码
6、cite：表示文本对某个文献的引用,应用的字体倾斜,一般指书籍,歌曲,电影等的标题;
7、datalist：下拉框,必须和input结合使用,list属性配合id名 


< input id=”mycar” list=”cars” >
< datalist id=”cars” >
    < option value=”跑车” >
    < option value=”自行车” >
    < option value=”轿车” >
< /datalist >

8、dialog：伪类,加open属性才会显示
9、del：定义删除文本
10、embed：引入一个插件，必须要有src属性
11、figure：定义图像，图表等，figcaption定义标题，两者搭配使用


< figure >
    < figcaption >图片标题< /figcaption >
< /figure >

12、fieldset：容纳一组表单元素，赋予特殊的边界效果，legend属性定义标题
13、ins：定义插入文本


< p >地方很< ins >的话< /ins >会觉< del >复合< /del >很机< /p>

14、keygen：提交给服务器一个秘钥
15、label：定义input标签，for属性和input标签的id属性相关联,当点击label中的文本时候，选中关联的input标签 


< label for=”username”>用户名
< input type=”text” id=”username”>

16、main：定义正文部分
17、menu：定义菜单
18、output：和input搭配是通用，输出input值
19、pre：定义多行代码
20、q：应用文字,文字加引号,一般引入短的语句
21、wbr：换行符，br标签是强制换行，wbr是可以换行，当空间不够时，就会换行
22、time：时分秒和年月日要用 T 分开 


< time datetime=”2016-06-17T15:25”>2016-06-17 15:25< /time >

这些标签：4，6，20是三种应用方式；5，19分别定义单行和多行代码；9、13分别为删除和插入文本。
