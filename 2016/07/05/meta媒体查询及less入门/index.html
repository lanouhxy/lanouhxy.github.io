@meta媒体查询及less入门

1、media媒体查询


1、在link中使用
< link rel=”stylesheet” type=”text/css” media=”only screen and (max-width:500px)” href=”2.css”>
当页面小于等于500px时候，调用2.css里的内容
2、在style中使用
< style >
@media only screen and (min-width:200px) and (max-width:400px){
    body{
    background:red;
    }
}
< /style >
当页面小于400px并且大于200px时候，body的背景为红色

2、less入门四种应用方法

1、变量：变量直接当做属性值引用，能够在复合属性中使用；
2、混合：混合将一个定义好的classA引用到classB中，实现classB继承classA的所有属性。
3、嵌套：我们可以在一个选择器中嵌套另一个选择器来实现继承。
4、函数和运算：运算除了提供加减乘除指望，可以实现属性值和颜色的运算。
