弹性布局及font-face

1、font-face变换字体


@font-face{
    font-family:”好看的华文行楷”;
    src:url(“fonts/华文行楷.ttf”);
}
p{
    font-family:”好看的华文行楷”;
    font-size:20px;
}

两个font-family要相同，要注意，有的一些特殊字体只能用于字母。

2、resize改变元素尺寸大小


div{
    width:100px;height:100px;border:1px solid black;
    resize:both;overflow:auto;
}
textarea{
    resize:horizontal;
}

resize:none/both/horizontal/vertical;四个值，规定不能/能/只能水平/只能垂直.
上面举了两个例子，一个是对起始不能改变的div标签，改变要加上overflow:auto;才能实现改变。另一个是对起始可以改变的textarea标签，直接设置resize属性。

3、column多列分隔

column-count:设置列的数目
column-gap:各列之间的间隔宽度
column-rule:定义列与列间隔线的样式


div{
    width:500px;height:500px;
    border:1px solid black;
    column-count:4;
    column-gap:50px;
    column-rule:5px solid red;
}

提示：column标签需要在前面加上webkit等的前缀。

4、弹性布局

display:box;使用弹性布局时候，必须给父级设置这个属性，只有设置这个属性，其子类才能使用下面的属性。
box-orient:horizontal/vertical;子类排列的方式，垂直还是水平。
box-flex:2;其子类占整个父类的分数。
box-direction:normal/reverse;子容器的排列顺序
box-align:start/center/end;子容器的垂直对齐方式
box-pack:start/center/end;子容器的水平对齐方式 


div{
    display:-webkit-box;
    -webkit-box-orient:horizontal;
}
div .span1{
    -webkit-box-flex:2;
}
div .span2{
    -webkit-box-flex:1;
}

在div中有两个span标签，水平排列，第一个占三分之二，第二个占三分之一。
