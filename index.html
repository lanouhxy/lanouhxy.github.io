过渡-变形-动画

1、transition过渡

1、transition:all 1.5s ease 1s;
transition-property:all/width/height/opacity,过渡的属性，不是所有属性都可以过渡，border大小就不可以过渡，而且过渡的属性，过渡前后的数值都需要写明；
transition-duration:过渡需要的时间，单位是s，可以为小数；
transition-timing-function:过渡变换的效率，ease/linear/ease-in等属性；
transition-delay:延迟时间，单位为s，可以为小数。
当transition写在hover中，移开鼠标，瞬间回到过渡之前的状态；当transition不写在hover中，移开鼠标，会反向变换回到之前的状态。

2、transform过渡

transform:rotate(0deg) scale(1) skew(50deg) translate(0px);
1、rotate()只有一个值，顺时针为正，逆时针为负；
2、scale(x,y)当只有一个值的时候，x与y同时变大，scaleX()是只在X方向变化；
3、skew(x,y)当只有一个值的时候，延X轴倾斜，skewX()绕x轴倾斜；
4、translate(x,y)当只有一个值的时候，延x轴偏移，translateX()延x轴偏移；

2、transform-3D过渡

1、rotateX:元素绕着X轴进行旋转；
rotateY:元素绕着Y轴进行旋转；
rotateZ：元素绕着Z轴进行旋转。
无论2d还是3d都可以实现X/Y/Z轴的旋转。
2、perspective:视距，视角，设置查看者的位置
transform-style:preserved-3d 设置这个属性后，所有的子元素都在3D空间呈现，设置这个属性就不能使用overflow:hidden
transform:tranlateZ();在设置perspective和transform-style属性之后，就可以看出随着Z轴移动，视角的变化，立体感。
3、rotate()属性无论是不是3d模式，都是可以绕X/Y/Z轴旋转的；
skew()属性没有绕Z轴倾斜的样式,即使设置3d模式，也没有绕Z轴倾斜的样式。

3、animation动画

1、animation-name:hxy 动画的名称，自己取
2、animation-duration:2s;动画的持续时间，单位为s
3、animation-timing-function:ease;动画的效果，变化速度
4、animation-iteration-count:infinite;动画循环次数，默认一次，infinite为无限循环播放
5、animation-delay:1s;动画延迟时间，单位为s
6、animation-direction:alternate;动画的播放方向，默认为normal，alternate动画播放在偶数次相反方向播放，奇数次正常方向播放。
7、animation-fill-mode:设置动画之外的状态，默认为none， forwards设置对象状态为动画结束时的状态；backwards设置对象状态为动画开始时的状态；both设置对象为动画结束或开始时的状态。
8、@keyframes hxy{}这个是定义动画的每一帧，可以用百分数表示。

圆角、阴影及背景颜色

1、圆角以及用圆角画图

border-radius:50% 20px 20px 20% / 20px 50% 20px 20px;
前面四个参数分别表示左上、右上、左下、右下四个角的水平方向，
后面四个参数分别表示左上、右上、左下、右下四个角的垂直方向，
左上水平对左上垂直，形成一个弧度。
简写方式：border-radius:50px;所有值都是50px

2、box-shadow/text-shadow

box-shadow:3px 3px 4px #fff inset;
五个参数分别代表X轴偏移量，Y轴偏移量，模糊程度，阴影颜色，内阴影。可以设置成负数，内阴影不设置时，就是正常阴影。
text-shadow:3px 3px 5px #ddd;
X轴偏移量、Y轴偏移量、模糊程度、阴影颜色，没有内阴影，偏移量可以设置成负数。

3、显示多个背景图片

css3中是可以在同一个元素中显示多个背景图片。
1、background:url(),url();引用背景图片，两个不同的图片。
2、background-repeat:no-repeat,no-repeat;两个no-repeat表明两种背景图片都不允许重复
3、background-position:0 0 ,right center;两个图片起始位置，一一对应
4、background-size:100px,contain;两个图片的大小，px/contain/cover；cover是宽高都铺满整个容器，不一定可以显示图片全貌；contain不一定铺满整个容器，又一遍铺满即可，显示图片全貌。
5、background-origin:padding-box/border-box/content-box;背景显示区域，padding-box是默认值，允许在padding区域显示背景图片；content-box是只允许在内容区域显示图片；border-box是允许在边框处显示图片。
6、background-clip:border-box/padding-box/content-box;背景图片允许显示区域，padding-box是默认值，允许在padding区域显示背景图片；content-box只允许在内容区域显示背景图片，border-box是允许在边框区域显示背景图片。
其中5和6两个属性，origin是图片显示区域，clip是允许图片显示的区域，如果origin定义为padding-box，而clip定义为content-box，那么图片只能显示content-box区域的部分，而padding区域的部分则不显示，就像被裁减一样。
7、background-image:linear-gradient(to left,black 20%,white 50%,blue)
背景颜色线性渐变：to left是从右向左渐变，black是从开始到20%处渐变成黑色，然后在20%到50%区域从黑色渐变成白色，最后在50%到100%区域从白色渐变成蓝色。
background:linear-gradient(to top,red 50%,green 50%)
background-size:50px 50px;
前者呈现一半红色，一半绿色，不渐变的效果，然后以50px为高度，重复呈现。
8、background-image:radial-gradient(ellipse at top,red 50px,black 150px,yellow)
背景颜色径向渐变：ellipse 变化呈椭圆变化，有圆和椭圆两种变化形式。at top 从上方开始渐变，如果不设置这个属性，则默认从中心点开始渐变；red 50px 从起始点处，以50px为半径为红色，然后50px到150px为红色到白色的渐变，150px到全图为白色到黄色的渐变。
9、background-image:repeating-radial-gradient(at left,red 20px,blue 50px)
冲左边开始，蓝色包裹红色，呈现大圆包小圆的全图模式。

css3选择器

1、子元素选择器（和后代选择器比较）

子元素选择器：形式E>F{}，选中的是对应的子元素，不包括子元素包含的子元素。
后代选择器：形式E F{}，选中是所有的后代，包括子元素里面的子元素。

2、兄弟选择器和通用选择器

兄弟选择器：形式E+F{}，1、必须是同级 2、必须相邻 3、必须在下方 4、只有一个符合要求
通用选择器：形式E~F{}, 1、必须同级 2、必须在下方 3、可以有多个符合要求

3、属性选择器

1.E[attr]：包含attr属性的E标签
2.E[attr=”value”]：属性attr=value的E标签
3.E[attr~=”value”]：表示几个空格隔开的属性值包含value
4.E[attr^=”value”]：表示以value开头的E标签
5.E[attr$=”value”]：表示以value结尾的E标签
6.E[attr*=”value”]：表示包含value值的E标签
7.E[attr|=”value”]：以value或value-开头的E标签
这些标签中，2和6标签比较常用

4、UI元素伪类选择器

disabled、enabled、checked、focus四种属性一般用在表单元素中,否定选择器一般和属相选择器配合使用。
1.input:focus{}选中含有focus属性的input标签，表现的形式
2.input:disabled{}有disabled属性的标签，不能选中
3.input:checked{}checked属性用在复选框里，选中表现的形式
4.input:enabled{}默认属性，可用的表单属性，与disabled属性相反
5.input:not([type=”sumbit”])用于不满足某类条件的标签

5、nth选择器

(1) first/last-child
E:first-child/last-child{}：选择E父级中的第一个/最后一个子元素，如果该元素是E元素，则表示形式
(2)nth-child
E:nth-child(n){}：选择E所在父级中第n个元素，如果该元素是E元素，则表现样式
E:nth-child(n+2){}：选择E所在父级中2及2以后的E元素
E:nth-child(-n+2){}：选择E所在父级中2及2之前的E元素
E:nth-child(2n+1){}：选择E所在父级中的奇数元素，如果是E元素，则表现样式
E:nth-child(2n){}：选择E所在父级中的偶数元素，如果是E元素，则表现样式
E:nth-child(odd){}：选择E所在父级中的奇数元素，如果是E元素，则表现样式
E:nth-child(even){}：选择E所在父级中的偶数元素，如果是E元素，则表现样式
(3)nth-of-type和nth-child相比较
E:nth-of-type(n){}：选择E所在的父级中第n个E元素
E:nth-child(n){}：选择E所在的父级中第n个元素，如果是E元素，则显示样式
E:first-of-type{}：选择E所在父元素中，第一个E元素
E:first-child{}：选择E所在父级中的第一个元素，结构上的第一个元素，如果是E元素，就表现样式
E:only-child{}：选择元素是它父级的唯一一个子元素，父级只包含一个子元素
E:only-of-type{}：选择元素是它父级元素的唯一一个相同类型的子元素
总的来说，nth-of-type是在一类元素中选择，nth-child是在所有元素中选择，再判断是否是所需要的E元素

6、伪元素

(1)::before/::after
E::before{}：在E元素的前面插入内容，
E::after{}：在E元素的后面插入内容，
这两种常和content配合使用，一般用来清除浮动，还有照片的icon
(2)::first-letter/::first-line
E::first-letter{}：选择文本的第一个字母或者汉字，主要用于段落排版
E::first-line{}：选择元素的第一行
(3)::selection
E::selection{}：用来改变浏览网页选中文本的默认效果，只能设置两个属性1.background 2.color

7、怪异盒模型box-sizing

(1)box-sizing:content-box：默认值，正常的盒模型
(2)box-sizing:border-box：这个盒模型，定义的宽高包括border和padding，改变padding和border的数值不会改变盒子的大小
(3)box-sizing:padding-box：盒模型的宽高包含padding，改变padding的数值不会改变盒子大小
(4)outline：只是绘制元素周围的一条线，位于边框边缘周围，起到突出元素的作用，轮廓线不占空间，也不一定是矩形

meta标签详解及添加ico图标

一、meta标签

meta标签，位于HTML文档的< head >与< title >之间，它提供的信息虽然用户不可见，但却是文档最基本的元信息。meta标签除了提供文档字符集、使用语言、作者等基本信息外，还涉及关键词和网页等级的设定。
meta标签的内容设计对搜索引擎优化（seo）来说是很重要的一个元素，合理运用Description和Keywords属性，可以优化网站。

1、< meta charset="UTF-8" > 字符编码
2、< meta http-equiv="X-UA-Compatible" content="IE=edge,chrome=1"/ > 优先识别IE最新版本和chrome 
3、< meta http-equiv="refresh" content="30";url=""/ > 规定每30秒刷新一次页面，如果加url，就会被重新定向到指定页面  
4、< meta name="renderer" content="webkit/ie-comp/ie-stand"/ > 内核控制meta标签，取用webkit内核，IE兼容内核，IE标准内核  
5、< meta name="description" content="120 words"/ > 对页面的描述，一般包括标点在内不超过120个词  
6、< meta name="keywords" content="袜子，鞋，电脑"/ > 定义文档关键词，用于搜索引擎，一般不超过12个名词   
7、< meta name="author" content="author name"/ > 网页制作的作者信息  
8、< meta name="robots" content="index,follow"/ >  
搜索引擎方式：  
（1）all：文件将被检索，并且页面上的链接也可以被查询；
（2）none：文件不能被检索，页面的链接也不被查询；
（3）index：文件将被检索；
（4）noindex：文件将不被检索；
（5）follow：页面的链接可以被查询；
（6）nofollow：页面的链接不可以被查询。

CSS3知识点梳理

CSS3第一天详解

主要知识点：

1.em和rem
2.max-width/min-width/max-height/min-height
3.vw/vh

1.em/rem

em:一个相对单位，相对于父级和自己的字号设置。把父类字号与自己字号相乘，得出字体最终字号。
rem:相对于根目录html，与父级或者body字号没有关联，只和根目录有关联，根目录的字号与自己字号相乘。
补充：字体大小可以设置成百分比，与父级字号有关。

2.max-width/max-height

max-width:元素的最大宽度，可以设置成百分比、像素、em/rem、vw/vh。
当设置成百分比的时候，相对于父级，占据父级的百分比。
min-width:元素的最小宽度，别的属性可以参照min-width。
max-height:元素的最大高度，可以参照max-width。
min-height:元素的最小高度，可以参照min-width。

3.vw/wh

vw:把视图的宽分为100份，整个视图是100vw，设置成50vw就是半个视图。
vh:把视图的高分为100份，整个视图是100vh，设置成50vh就是半个视图。

总结

今天重点讲了一些弹性布局，随着页面的放大缩小，页面里面的元素也跟着放大缩小，这可以通过给元素设置百分比、em、vw实现。

创建博客

具体步骤

1、下载并安装NodeJS

(1)在node官网下载NodeJS.
(2)测试NodeJS，输入命令“node -v”，当显示v+数字组合时候就表明安装成功。

2、安装hexo

(1)先输入命令“hexo install -g hexo”，如果出现报错，则尝试输入命令”sudo hexo install -g hexo”.
(2)输入管理员密码进行身份验证。
(3)输入命令“hexo version”，出现hexo相关信息则表明安装成功。
(4)输入命令“cd Desktop”,然后输入“mkdir 博客名称”在桌面创建一个存放博客的文件夹。
(5)输入命令“hexo init”初始化hexo目录，然后输入“hexo server”命令，运行本地配置。
(6)复制出现的网址，打开就可以看到之前所建的博客。

3、配置Github

(1)申请一个github账号。
(2)配置ssh-key，先输入命令“gif”，然后三次回车，接着输入“vim ~/.ssh/id_rsa.pub”，会出现一个对话框，复制对话框里面的内容，然后输入“:q”退出窗口，回到起初命令行。
(3)登陆申请的Github账号，设置其中的-SSH and GPG keys-New SSH key,把第二步骤中复制的代码添加进去。
(4)配置Github,输入命令“git config - -global user.name 申请的Github的用户名”，然后再输入命令“git config - -global user.email 申请的Github邮箱”。
(5)点击“new repository”新建一个域名仓库，输入“用户名.github.io”,然后勾选“Initialize this repository with a README”,提交即可完成。

4、发布博客

(1)打开之前创建的博客文件夹，找到“Sublime Text”打开其中的“_config.yml”文件。
(2)在文档的最下面，修改“type: git”和添加“repo: 域名仓库的https地址”。
(3)新建微博，输入命令“hexo new 微博名称”,输入命令“hexo g”转换文本格式，再输入命令“hexo d”把文本上传到服务器。
(4)登陆浏览器，查看微博。
